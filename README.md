# 📊 Bank Profitability Analytics Dashboard

## 📌 Project Overview
This project is a **Streamlit-based web application** developed to analyze and predict **bank profitability (ROA - Return on Assets)** using key financial indicators.

The dashboard provides insights into how factors like NPA, CAR, Credit Growth, and Cost Efficiency impact bank performance.

---

## 🎯 Objective
- To analyze bank financial performance  
- To predict **ROA (profitability)** using Machine Learning  
- To build an interactive dashboard for decision-making  

---

## 📂 Dataset
The dataset contains information about:
- Bank Name
- Year
- Net NPA Ratio (%)
- CAR / CRAR (%)
- Credit Growth (%)
- Cost-to-Income Ratio (%)
- Bank Size (Log Assets)
- ROA (%) (Target Variable)

---

## 🤖 Model Used
- Machine Learning Model: **Regression Model**
- Evaluation Metric: **R² Score**

---

## 📊 Features of the App
- 📈 Interactive Dashboard  
- 📊 Bank-wise ROA Trend Analysis  
- 📉 Key Financial Metrics Display  
- 🤖 ROA Prediction using user inputs  

---

## 🛠️ Technologies Used
- Python  
- Streamlit  
- Pandas  
- NumPy  
- Scikit-learn  
- Plotly
